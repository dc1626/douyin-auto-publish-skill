# Douyin Auto Publish Skill

抖音自动发布技能 - 帮你自动运营抖音账号。

## 功能

- 🔍 **自动选题**：从 GitHub Releases、Moltbook 等平台获取热门内容
- ✍️ **内容创作**：整理成抖音风格的文案（25字内标题 + 正文 + 话题）
- ✅ **审核机制**：先发给你审核，确认后再发布
- 📤 **自动发布**：审核通过后自动发布到抖音
- 📝 **记录归档**：发布成功后会记录到 memory

## 使用方法

1. 安装此 skill 后，告诉 AI"帮我发一条抖音"
2. AI 会自动选题并创作内容
3. 内容发给你审核
4. 你确认后 AI 自动发布
5. 发布成功后通知你
